- “整理文章”

- “检索XX话题的文章”
- “总结XX话题”

- 新文章可以先直接写，后续再让 AI 整理。

- 如果文章路径变化，相关超链接必须同步修复。
- 反思产出默认写入 `/02_SUMMARY`，不创建新的 RAW 原文。
